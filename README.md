**Face & Hand Landmark Recognition with Gesture Detection**

This project is a real-time face and hand landmark recognition system that uses a desktop webcam to detect human presence,
analyze hand gestures, and identify whether the screen is empty or occupied. 
The application also recognizes common hand gestures and displays the corresponding sign.

**Features**

> Real-time webcam-based detection

> Face landmark detection

> Hand landmark tracking

> Hand gesture recognition

> Detects whether the screen is empty or occupied

> Identifies hand signs such as:

**How It Works**

>Only hands present
